# Paper Reviews Guide

This directory contains the canonical review folders for papers that have been investigated.

Store reviews under:

- `docs/papers/reviews/YYYY/YYYY_author_shorttitle/review.md`
- `docs/papers/reviews/YYYY/YYYY_author_shorttitle/paper.pdf`

One folder per paper keeps room for future assets, extracted figures, reproduction notes, and the original paper PDF without cluttering the top level.

## Expected review folder contents

- `review.md`: canonical Hermes review
- `paper.pdf`: downloaded source PDF for local reading and traceability

When a PDF can be downloaded from the paper source, store it as `paper.pdf`. If redistribution rights are unclear and the repository later becomes public, revisit whether the file should remain versioned.

Use the template at `docs/papers/_templates/review.md`.
