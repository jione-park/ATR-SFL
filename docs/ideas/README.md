# Idea Notes Guide

아이디어 노트는 최종 주장 문서가 아니라, 잠재적인 contribution 후보를 구조적으로 정리하는 문서다.

앞으로 `docs/ideas/*.md`의 정식 본문은 기본적으로 한국어로 작성한다.

- 모델명
- theorem notation
- 논문 제목
- 꼭 필요한 기술 용어

는 영어를 그대로 써도 된다.

각 idea note는 아래 질문에 답할 수 있어야 한다.

- 어떤 문제를 겨냥하는가
- 왜 작동할 가능성이 있는가
- 어떻게 실패할 수 있는가
- 어떤 ablation이 필요한가
- 계속 투자할 만큼의 근거는 무엇인가

## File naming

Use `YYYYMMDD_short_idea_name.md`.
