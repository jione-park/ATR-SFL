# Source Guide

Put implementation code here.

Current bootstrap layout:

- `src/models/`
- `src/data/`
- `src/engine/`
- `src/utils/`
- `src/train.py`

Current first target:

- `CIFAR-10`
- `IID`
- `full-token`
- `DeiT-Tiny`
- `training-time SFL`

Keep experimental one-off scripts out of `src/` unless they are on a path to becoming part of the reproducible stack.
